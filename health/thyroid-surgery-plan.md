## 🏃 แผน 3 เดือนก่อนผ่าตัด

### 📅 เดือนที่ 1: มีนาคม 2026 (ตอนนี้!)

#### 💪 สุขภาพกาย
- [ ] ควบคุมความดันโลหิตให้คงที่
- [ ] ควบคุมไขมันในเลือด
- [ ] เริ่มออกกำลังกายเบาๆ (เดิน 30 นาที/วัน)
- [ ] ปรับอาหาร - ลดเค็ม ลดหวาน ลดมัน
- [ ] นอนให้เพียงพอ (7-8 ชม.)
- [ ] เลิกบุหรี่ (ถ้าสูบ)
- [ ] งดแอลกอฮอล์

#### 🧘 สุขภาพใจ
- [ ] นั่งสมาธิทุกวัน (เช้า/เย็น)
- [ ] ทำสิ่งที่ชอบ ลดความเครียด
- [ ] คุยกับคนใกล้ชิดเรื่องความกังวล

#### 💼 งาน/ธุระ
- [ ] แจ้งที่ทำงานเรื่องวันลา
- [ ] วางแผนงานที่ต้องส่งมอบ
- [ ] จัดเอกสารสำคัญให้เรียบร้อย

#### 💰 การเงิน
- [ ] เตรียมเงินสำรองสำหรับค่ารักษา
- [ ] ตรวจสอบสิทธิ์ประกันสังคม/ประกันสุขภาพ
- [ ] ตรวจสอบสิทธิ์ที่ รพ.ศิริราช

---

### 📅 เดือนที่ 2: เมษายน 2026

#### 💪 สุขภาพกาย
- [ ] ตรวจเลือดตามนัดแพทย์
- [ ] เพิ่มโปรตีนในอาหาร (ปลา ไก่ ไข่)
- [ ] วิตามินซี - ผลไม้รสเปรี้ยว
- [ ] ออกกำลังกายต่อเนื่อง

#### 🧘 สุขภาพใจ
- [ ] ฝึกหายใจลึก (ใช้หลังผ่าตัด)
- [ ] ศึกษาข้อมูลการผ่าตัด (รู้ลึก = ลดกังวล)
- [ ] เขียนบันทึกความรู้สึก

#### 💼 งาน/ธุระ
- [ ] ส่งมอบงานที่ค้างอยู่
- [ ] ฝึกคนรับงานแทน (ถ้ามี)
- [ ] เตรียมงานล่วงหน้าสำหรับช่วงพักฟื้น

#### 🏠 ที่พัก
- [ ] วางแผนที่พักระหว่างรักษา (ใกล้ รพ.?)
- [ ] เตรียมของใช้สำหรับอยู่โรงพยาบาล

---

### 📅 เดือนที่ 3: พฤษภาคม 2026

#### 💪 สุขภาพกาย
- [ ] ตรวจร่างกายก่อนผ่าตัด (Pre-op)
- [ ] ดูแลสุขภาพพิเศษ - ไม่ป่วยเด็ดขาด!
- [ ] รับประทานอาหารให้ครบ 5 หมู่
- [ ] พักผ่อนให้เพียงพอ

#### 🧘 สุขภาพใจ
- [ ] ทำใจยอมรับ
- [ ] เตรียมใจสำหรับการผ่าตัด
- [ ] วางแผนชีวิตหลังผ่าตัด

#### 💼 งาน/ธุระ
- [ ] สรุปงานทั้งหมด
- [ ] แจ้งเพื่อนร่วมงานเรื่องวันหยุด
- [ ] มอบหมายงานชั่วคราว

---

### 📅 2 สัปดาห์สุดท้าย: มิถุนายน 2026

#### 🏥 เตรียมตัวสุดท้าย
- [ ] ปรึกษาแพทย์เรื่องยาที่ทานอยู่
- [ ] หยุดยาบางชนิดตามคำแพทย์ (เช่น ยาละลายลิ่มเลือด)
- [ ] เตรียมเอกสาร: บัตรประชาชน, ประกันสังคม, ใบรับรองแพทย์
- [ ] เตรียมของใช้ส่วนตัวสำหรับอยู่โรงพยาบาล
- [ ] อาบน้ำทำความสะอาดก่อนไปโรงพยาบาล

#### 📋 ของที่ต้องเตรียม
- [ ] เสื้อผ้าสบายๆ (เปิดหน้าอกง่าย)
- [ ] ของใช้ส่วนตัว (แปรงสีฟัน ผ้าเช็ดตัว)
- [ ] โทรศัพท์ + ชาร์จ
- [ ] เอกสารสำคัญ
- [ ] เงินสดสำรอง

## ⚠️ สิ่งที่ต้องระวังพิเศษ

### เนื่องจากก้อนใหญ่และอาจต้องเปิดกระดูกหน้าอก:

1. **เลือกทีมแพทย์ที่มีประสบการณ์** ✅ (ศิริราช)
2. **เตรียมหน่วยเลือดสำรอง** (ถ้าจำเป็น)
3. **วางแผนพักฟื้นยาวขึ้น** (อาจ 2-4 สัปดาห์)
4. **เตรียมผู้ดูแล** (ช่วงแรกอาจขยับตัวลำบาก)
