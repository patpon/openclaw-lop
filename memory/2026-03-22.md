## Technical Notes
- When fixing HTML files, always check for:
  1. Duplicate closing tags (`</html>`, `</body>`, `</script>`)
  2. Garbage text at end of file
  3. Missing JavaScript functions referenced in HTML onclick attributes
- Use `grep -c '</html>' filename` to find duplicate closing tags
- Use `tail -20 filename` to inspect end of files for corruption
