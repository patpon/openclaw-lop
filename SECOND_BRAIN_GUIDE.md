# วิธีการหา Second Brain System ใน Dashboard

## 📍 ตำแหน่งที่อยู่:
1. **เข้า Dashboard:** https://ajpatpon.cloud
2. **Login:** admin / openclaw2024
3. **อยู่ในหน้าแรก (Overview)**
4. **เลื่อนลงมาด้านล่างสุด**

## 🎯 หากยังไม่เห็น ให้ลอง:

### **Step 1: ตรวจสอบว่า Login สำเร็จ**
- ดูมุมขวาบน จะต้องมีตัวอักษร "Online" สีเขียว
- ถ้าไม่มี ให้ Login ใหม่

### **Step 2: เลื่อนลงมาด้านล่าง**
- **ใช้ลูกศรลง** หรือ **เลื่อน Mouse wheel**
- **หาหัวข้อที่ชื่อ:** "ระบบสมองที่สอง - Quick Access"
- **จะอยู่ใต้ส่วนของ Activity Chart**

### **Step 3: ทดสอบปุ่มเปิด**
- **คลิกปุ่ม "เปิด"** ในการ์ด "ไอเดีย"
- **จะต้องมี Popup ขึ้นมา** แสดงเนื้อหา

## 🔧 หากยังไม่ได้ ให้ลอง:

### **Clear Browser Cache:**
- **Chrome:** Ctrl+Shift+R
- **Firefox:** Ctrl+F5
- **Safari:** Cmd+Shift+R

### **Check Developer Console:**
- **กด F12** เปิด Developer Tools
- **ไปแท็บ Console**
- **ดูว่ามี error อะไรหรือไม่**

### **Try Different Browser:**
- **เปลี่ยน Browser** ชั่วคราว
- **หรือเปิดใน Private/Incognito Mode**

## 📞 หากยังไม่ได้ บอกน้องได้เลย:
- **ส่ง Screenshot** มาให้ดู
- **บอกว่าหน้าเว็บแสดงอะไร**
- **บอกว่ามีข้อความ error อะไร**
