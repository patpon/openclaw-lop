# KNOWLEDGE.md - ความรู้สะสม

## ความรู้เกี่ยวกับเทคโนโลยี

### 💻 การเขียนโค้ด
- **HTML/CSS/JavaScript** - พื้นฐานการสร้างเว็บแอปพลิเคชัน
- **Node.js + Express** - สร้าง backend server และ API
- **Bootstrap 5** - Framework CSS สำหรับการออกแบบที่สวยงาม
- **Chart.js** - สร้างกราฟและแผนภูมิต่างๆ

### 🛡️ ความปลอดภัย
- **UFW Firewall** - กำหนดกฎการป้องกันเบื้องต้น
- **Fail2Ban** - ป้องกันการโจมตีแบบ brute force
- **Nginx Reverse Proxy** - ทำ load balancing และ SSL termination
- **Systemd Services** - จัดการ services บน Linux

### 🚀 OpenClaw ระบบ
- **Gateway** - จัดการการเชื่อมต่อและ device pairing
- **Agents** - AI assistants ที่ทำงานแบบ isolated
- **Cron Jobs** - ตั้งเวลางานอัตโนมัติ
- **Memory System** - จัดการความจำและบริบท

## ความรู้เกี่ยวกับการออกแบบ

### 🎨 หลักการออกแบบ UI/UX
- **User-Centric Design** - ออกแบบให้ผู้ใช้เป็นศูนย์กลาง
- **Mobile First** - ออกแบบให้รองรับมือถือก่อน
- **Accessibility** - ทำให้คนพิการใช้งานได้
- **Performance** - เน้นความเร็วในการโหลด

### 📊 การแสดงข้อมูล
- **Data Visualization** - แสดงข้อมูลเป็นกราฟและแผนภูมิ
- **Timeline Interface** - แสดงกิจกรรมเป็นเวลา
- **Card-based Layout** - จัดข้อมูลเป็นการ์ดที่อ่านง่าย
- **Responsive Design** - ปรับขนาดตามหน้าจอ
